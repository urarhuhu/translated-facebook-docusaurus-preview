# 圖表繪製

可使用 [Mermaid](https://mermaid-js.github.io/mermaid/) 在代碼區塊中渲染圖表。

## 安裝方式 {#installation}

```bash npm2yarn
npm install --save @docusaurus/theme-mermaid
```

在 `docusaurus.config.js` 中添加插件 `@docusaurus/theme-mermaid` 並將 `markdown.mermaid` 設為 `true` 以啟用 Mermaid 功能。

```js title="docusaurus.config.js"
export default {
  markdown: {
    mermaid: true,
  },
  themes: ['@docusaurus/theme-mermaid'],
};
```

## 使用方法 {#usage}

添加語言標記為 `mermaid` 的代碼區塊：

````md title="Example Mermaid diagram"
```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```
````

```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```

詳見 [Mermaid 語法文檔](https://mermaid-js.github.io/mermaid/#/./n00b-syntaxReference) 以獲取更多 Mermaid 語法資訊。

## 主題設定 {#theming}

可通過在 `docusaurus.config.js` 的 `themeConfig` 中設置 `mermaid.theme` 值來變更圖表的深色與淺色主題，同時支援為兩種模式分別設定主題。

```js title="docusaurus.config.js"
export default {
  themeConfig: {
    mermaid: {
      theme: {light: 'neutral', dark: 'forest'},
    },
  },
};
```

詳見 [Mermaid 主題文檔](https://mermaid-js.github.io/mermaid/#/theming) 以獲取更多主題設定資訊。

## Mermaid 配置 {#configuration}

`mermaid.options` 中的配置將直接傳遞給 `mermaid.initialize`：

```js title="docusaurus.config.js"
export default {
  themeConfig: {
    mermaid: {
      options: {
        maxTextSize: 50,
      },
    },
  },
};
```

詳見 [Mermaid 配置文檔](https://mermaid-js.github.io/mermaid/#/./Setup?id=configuration) 及 [Mermaid 配置類型](https://github.com/mermaid-js/mermaid/blob/master/packages/mermaid/src/config.type.ts) 以獲取可用配置選項。

## 動態 Mermaid 組件 {#component}

如需生成動態圖表，可使用 `Mermaid` 組件：

```mdx title="Example of dynamic Mermaid component"
import Mermaid from '@theme/Mermaid';

<Mermaid
  value={`graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;`}
/>
```
