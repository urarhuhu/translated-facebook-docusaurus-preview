# 遊樂場

遊樂場讓您能在瀏覽器中直接運行 Docusaurus，無需安裝任何軟體！

主要適用於以下場景：

- 測試 Docusaurus
- 回報錯誤

可使用易記的快捷網址 [docusaurus.new](https://docusaurus.new)。

請從下方選擇一個可用選項。

```mdx-code-block
import {PlaygroundCardsRow} from '@site/src/components/Playground';

<PlaygroundCardsRow />
```

:::tip

為方便起見，下次造訪 [docusaurus.new](https://docusaurus.new) 時會記住您的選擇。

:::
