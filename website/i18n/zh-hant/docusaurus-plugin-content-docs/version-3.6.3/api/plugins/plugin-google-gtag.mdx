# 📦 plugin-google-gtag

import APITable from '@site/src/components/APITable';

預設的 [全域網站代碼 (gtag.js)](https://developers.google.com/tag-platform/gtagjs) 外掛程式。這是一個 JavaScript 標記框架與 API，可讓您將事件資料傳送至 Google Analytics、Google Ads 和 Google Marketing Platform。本節說明如何設定 Docusaurus 網站以啟用 Google Analytics 的全域網站代碼。

:::tip

您可以使用 [Google 的標記助理工具](https://tagassistant.google.com/) 來檢查 gtag 是否設定正確！

:::

:::warning[僅限正式環境]

此外掛程式在開發環境中始終處於非活動狀態，並**僅在正式環境中啟用**，以避免污染分析統計數據。

:::

## 安裝 {#installation}

```bash npm2yarn
npm install --save @docusaurus/plugin-google-gtag
```

:::tip

如果您使用 `@docusaurus/preset-classic` 預設設定，則無需將此外掛程式安裝為依賴項。

您可以透過 [預設選項](../../using-plugins.mdx#docusauruspreset-classic) 來設定此外掛程式。

:::

## 設定 {#configuration}

可接受的欄位：

```mdx-code-block
<APITable>
```

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `trackingID` | <code>string \| string[]</code> | **Required** | The tracking ID of your gtag service. It is possible to provide multiple ids. |
| `anonymizeIP` | `boolean` | `false` | Whether the IP should be anonymized when sending requests. |

</APITable>

### 範例設定 {#ex-config}

您可以透過預設選項或外掛程式選項來設定此外掛程式。

:::tip

大多數 Docusaurus 使用者會透過預設選項來設定此外掛程式。

:::

```js config-tabs
// Preset Options: gtag
// Plugin Options: @docusaurus/plugin-google-gtag

const config = {
  trackingID: 'G-999X9XX9XX',
  anonymizeIP: true,
};
```
