# 遊樂場

遊樂場讓您能在瀏覽器中直接運行 Docusaurus，無需安裝任何軟體！

這功能主要適用於：

- 測試 Docusaurus
- 回報錯誤

可使用簡易好記的快捷網址 [docusaurus.new](https://docusaurus.new)。

請從以下選項中擇一使用。

```mdx-code-block
import {PlaygroundCardsRow} from '@site/src/components/Playground';

<PlaygroundCardsRow />
```

:::tip

為方便起見，系統會記住您的選擇，下次造訪 [docusaurus.new](https://docusaurus.new) 時將自動套用。

:::
