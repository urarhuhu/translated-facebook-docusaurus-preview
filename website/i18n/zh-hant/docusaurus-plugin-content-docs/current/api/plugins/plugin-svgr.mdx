# 📦 plugin-svgr

import APITable from '@site/src/components/APITable';

一個 [SVGR](https://react-svgr.com/) 插件，可在建置時自動將 SVG 檔案轉換為 React 元件。

## 安裝 {#installation}

```bash npm2yarn
npm install --save @docusaurus/plugin-svgr
```

:::tip

若您使用 `@docusaurus/preset-classic` 預設集，則無需將此插件安裝為依賴項。

您可以透過[預設集選項](../../using-plugins.mdx#docusauruspreset-classic)來配置此插件。

:::

## 配置 {#configuration}

可接受欄位：

```mdx-code-block
<APITable>
```

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `svgrConfig` | `object` | `{}` | The [SVGR config options](https://react-svgr.com/docs/options/), forwarded as is |

```mdx-code-block
</APITable>
```

### 範例配置 {#ex-config}

您可以透過插件選項來配置此插件。

```js config-tabs
// Preset Options: svgr
// Plugin Options: @docusaurus/plugin-svgr

const config = {
  svgrConfig: {
    /* SVGR config */
  },
};
```
