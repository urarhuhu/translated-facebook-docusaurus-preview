# 升級 Docusaurus

Docusaurus 的版本控制基於 `major.minor.patch` 方案，並遵循 [**語意化版本控制**](https://semver.org/)。

**重大變更**僅在主要版本升級時發布，並詳細記錄在以下升級指南中。

## 升級疑難排解

升級 Docusaurus 時，您可能會遇到因快取依賴項不匹配而導致的問題 — 在回報錯誤或尋求支援之前，您應執行以下幾個疑難排解步驟來解決這些常見問題。

### 執行 `clear` 指令

此 CLI 指令用於清除 Docusaurus 網站生成的資源、快取和建構產物。

```bash npm2yarn
npm run clear
```

### 移除 `node_modules` 和鎖定檔案

使用以下指令移除 `node_modules` 資料夾和您的套件管理器的鎖定檔案：

<Tabs>
<TabItem label="Bash" value="bash">

```bash
rm -rf node_modules yarn.lock package-lock.json
```

</TabItem>
<TabItem label="PowerShell" value="powershell">

```powershell
@('node_modules','yarn.lock','package-lock.json') | Remove-Item -Force -Recurse -ErrorAction SilentlyContinue
```

</TabItem>
</Tabs>

然後重新安裝套件並重新生成 `lock` 檔案：

```bash npm2yarn
npm install
```
