# no-untranslated-text

import APITable from '@site/src/components/APITable';

強制要求 JSX 中的文字標籤必須透過翻譯函式呼叫進行包裹。

當使用[國際化功能](../../../i18n/i18n-introduction.mdx)時，此規則確保網站上顯示的所有標籤皆可翻譯，避免任何字串意外未經翻譯直接顯示。

## 規則詳情 {#details}

此規則的**錯誤**程式碼範例：

```js
// Hello World is not translated
<Component>Hello World</Component>
```

此規則的**正確**程式碼範例：

```js
// Hello World is translated
<Component>
  <Translate>Hello World</Translate>
</Component>
```

## 規則配置 {#configuration}

可接受欄位：

```mdx-code-block
<APITable>
```

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `ignoredStrings` | `string[]` | `[]` | Text labels that only contain strings in this list will not be reported. |

```mdx-code-block
</APITable>
```

## 不適用情境 {#when-not-to-use}

若未使用[國際化功能](../../../i18n/i18n-introduction.mdx)，可停用此規則。亦可在文字無需翻譯的情境下停用本規則。
