# 📦 plugin-google-tag-manager

import APITable from '@site/src/components/APITable';

用於將 [Google Tag Manager (gtm.js)](https://developers.google.com/tag-platform/tag-manager) 添加到 Docusaurus 網站的插件。可與標準的 [gtag 插件](./plugin-google-gtag.mdx) 配合使用，深入分析用戶如何使用您的網站。

:::tip

您可以使用 [Google 的 Tag Assistant](https://tagassistant.google.com/) 工具檢查標籤管理器的設定是否正確！

:::

:::warning[僅限正式環境]

此插件在開發環境中始終處於非活動狀態，且**僅在正式環境中啟用**，以避免污染分析統計數據。

:::

## 安裝 {#installation}

```bash npm2yarn
npm install --save @docusaurus/plugin-google-tag-manager
```

:::tip

如果您使用預設配置 `@docusaurus/preset-classic`，則無需將此插件安裝為依賴項。

您可以通過 [預設選項](../../using-plugins.mdx#docusauruspreset-classic) 配置此插件。

:::

### 配置範例 {#ex-config}

您可以通過預設選項或插件選項配置此插件。

:::tip

大多數 Docusaurus 用戶通過預設選項配置此插件。

:::

```js config-tabs
// Preset Options: googleTagManager
// Plugin Options: @docusaurus/plugin-google-tag-manager
const config = {
  containerId: 'GTM-12345',
};
```
