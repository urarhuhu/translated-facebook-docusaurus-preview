# 圖表繪製

您可以使用 [Mermaid](https://mermaid-js.github.io/mermaid/) 在代碼區塊中渲染圖表。

## 安裝方法 {#installation}

```bash npm2yarn
npm install --save @docusaurus/theme-mermaid
```

在 `docusaurus.config.js` 中添加插件 `@docusaurus/theme-mermaid` 並將 `markdown.mermaid` 設為 `true` 以啟用 Mermaid 功能。

```js title="docusaurus.config.js"
module.exports = {
  markdown: {
    mermaid: true,
  },
  themes: ['@docusaurus/theme-mermaid'],
};
```

## 使用方式 {#usage}

添加語言標記為 `mermaid` 的代碼區塊：

````md title="Example Mermaid diagram"
```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```
````

```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```

詳見 [Mermaid 語法文檔](https://mermaid-js.github.io/mermaid/#/./n00b-syntaxReference) 以獲取更多關於 Mermaid 語法的資訊。

## 主題設定 {#theming}

可透過在 `docusaurus.config.js` 的 `themeConfig` 中設置 `mermaid.theme` 值來變更圖表的深色與淺色主題。您可分別設定淺色與深色模式的主題。

```js title="docusaurus.config.js"
module.exports = {
  themeConfig: {
    mermaid: {
      theme: {light: 'neutral', dark: 'forest'},
    },
  },
};
```

詳見 [Mermaid 主題文檔](https://mermaid-js.github.io/mermaid/#/theming) 以獲取更多關於 Mermaid 圖表主題設定的資訊。

## Mermaid 配置 {#configuration}

`mermaid.options` 中的配置選項將直接傳遞給 `mermaid.initialize`：

```js title="docusaurus.config.js"
module.exports = {
  themeConfig: {
    mermaid: {
      options: {
        maxTextSize: 50,
      },
    },
  },
};
```

詳見 [Mermaid 配置文檔](https://mermaid-js.github.io/mermaid/#/./Setup?id=configuration) 以查看可用的配置選項。
