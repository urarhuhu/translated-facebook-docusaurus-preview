# 遊樂場

遊樂場讓您能夠**直接在瀏覽器中運行 Docusaurus，無需安裝任何東西**！

這主要適用於：

- 測試 Docusaurus
- 回報錯誤

使用 [docusaurus.new](https://docusaurus.new) 作為容易記住的快捷方式。

請從以下可用選項中選擇一個。

```mdx-code-block
import {PlaygroundCardsRow} from '@site/src/components/Playground';

<PlaygroundCardsRow />
```

:::tip

為方便起見，我們會記住您的選擇，下次您訪問 [docusaurus.new](https://docusaurus.new) 時會自動套用。

:::
