# 📦 theme-search-algolia

此主題提供了一個與 Algolia DocSearch 輕鬆整合的 `@theme/SearchBar` 組件。結合 `@docusaurus/theme-classic` 使用時，可實現極簡的搜索整合功能。更多資訊請參閱[搜索](../../search.mdx)文檔。

```bash npm2yarn
npm install --save @docusaurus/theme-search-algolia
```

此主題還新增了位於 `/search` 路徑的可搜索頁面（作為可替換的 `SearchPage` 組件），並支援 OpenSearch。您可通過 `themeConfig.algolia.searchPagePath` 修改默認路徑，設置為 `false` 則禁用搜索頁面。

:::tip

若已安裝 `@docusaurus/preset-classic`，則無需將其作為依賴項單獨安裝。

:::
