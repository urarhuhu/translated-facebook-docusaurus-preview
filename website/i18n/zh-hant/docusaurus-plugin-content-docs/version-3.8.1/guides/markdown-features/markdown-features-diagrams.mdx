# 圖表

您可以在程式碼區塊中使用 [Mermaid](https://mermaid-js.github.io/mermaid/) 來渲染圖表。

## 安裝 {#installation}

```bash npm2yarn
npm install --save @docusaurus/theme-mermaid
```

在您的 `docusaurus.config.js` 中，新增插件 `@docusaurus/theme-mermaid` 並將 `markdown.mermaid` 設為 `true`，以啟用 Mermaid 功能。

```js title="docusaurus.config.js"
export default {
  markdown: {
    mermaid: true,
  },
  themes: ['@docusaurus/theme-mermaid'],
};
```

## 使用方法 {#usage}

新增一個語言標記為 `mermaid` 的程式碼區塊：

````md title="Example Mermaid diagram"
```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```
````

```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```

有關 Mermaid 語法的更多資訊，請參閱 [Mermaid 語法文件](https://mermaid-js.github.io/mermaid/#/./n00b-syntaxReference)。

## 主題設定 {#theming}

您可以透過在 `docusaurus.config.js` 的 `themeConfig` 中設定 `mermaid.theme` 值來變更圖表的深色與淺色主題。您可以分別為淺色與深色模式設定主題。

```js title="docusaurus.config.js"
export default {
  themeConfig: {
    mermaid: {
      theme: {light: 'neutral', dark: 'forest'},
    },
  },
};
```

有關 Mermaid 圖表主題設定的更多資訊，請參閱 [Mermaid 主題文件](https://mermaid-js.github.io/mermaid/#/theming)。

## Mermaid 設定 {#configuration}

`mermaid.options` 中的選項將直接傳遞給 `mermaid.initialize`：

```js title="docusaurus.config.js"
export default {
  themeConfig: {
    mermaid: {
      options: {
        maxTextSize: 50,
      },
    },
  },
};
```

有關可用的設定選項，請參閱 [Mermaid 設定文件](https://mermaid-js.github.io/mermaid/#/./Setup?id=configuration) 與 [Mermaid 設定類型](https://github.com/mermaid-js/mermaid/blob/master/packages/mermaid/src/config.type.ts)。

## 動態 Mermaid 元件 {#component}

若要生成動態圖表，您可以使用 `Mermaid` 元件：

```mdx title="Example of dynamic Mermaid component"
import Mermaid from '@theme/Mermaid';

<Mermaid
  value={`graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;`}
/>
```
