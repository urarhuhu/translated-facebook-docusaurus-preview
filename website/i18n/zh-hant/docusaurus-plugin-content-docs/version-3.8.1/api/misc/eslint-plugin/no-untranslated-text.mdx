# no-untranslated-text

import APITable from '@site/src/components/APITable';

強制要求 JSX 中的文字標籤必須使用 translate 調用進行包裹。

當使用 [i18n 功能](../../../i18n/i18n-introduction.mdx) 時，此規則確保網站上顯示的所有標籤都是可翻譯的，避免任何字串意外未被翻譯。

## 規則詳情 {#details}

此規則的**錯誤**程式碼範例：

```js
// Hello World is not translated
<Component>Hello World</Component>
```

此規則的**正確**程式碼範例：

```js
// Hello World is translated
<Component>
  <Translate>Hello World</Translate>
</Component>
```

## 規則配置 {#configuration}

可接受的欄位：

```mdx-code-block
<APITable>
```

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `ignoredStrings` | `string[]` | `[]` | Text labels that only contain strings in this list will not be reported. |

```mdx-code-block
</APITable>
```

## 不適用情況 {#when-not-to-use}

如果您沒有使用 [i18n 功能](../../../i18n/i18n-introduction.mdx)，可以禁用此規則。對於不需要翻譯的文字，也可以禁用此規則。
