# 📦 theme-search-algolia

此主題提供了一個與 Algolia DocSearch 輕鬆整合的 `@theme/SearchBar` 組件。結合 `@docusaurus/theme-classic` 使用時，可實現極簡的搜尋功能整合。更多資訊請參閱[搜尋功能](../../search.mdx)文件。

```bash npm2yarn
npm install --save @docusaurus/theme-search-algolia
```

本主題還新增了位於 `/search` 路徑的搜尋頁面（可透過 swizzle 調整的 `SearchPage` 組件），並支援 OpenSearch 協議。您可透過 `themeConfig.algolia.searchPagePath` 修改預設路徑，設為 `false` 則會停用搜尋頁面。

:::tip

若已安裝 `@docusaurus/preset-classic`，則無需額外安裝本主題作為依賴項。

:::
