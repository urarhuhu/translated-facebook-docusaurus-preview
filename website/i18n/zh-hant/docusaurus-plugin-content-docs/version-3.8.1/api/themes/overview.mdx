# Docusaurus 主題

我們提供官方的 Docusaurus 主題。

## 主要主題 {#main-themes}

主要主題為 [文檔](../plugins/plugin-content-docs.mdx)、[部落格](../plugins/plugin-content-blog.mdx) 和 [頁面](../plugins/plugin-content-pages.mdx) 插件實現了使用者介面。

- [@docusaurus/theme-classic](./theme-classic.mdx)
- 🚧 其他主題規劃中

:::warning

目標是讓所有主題共享完全相同的功能、使用者體驗和配置。

僅 UI 設計和底層樣式框架應有所不同，且您應能輕鬆更換主題。

目前尚未達成此目標：僅經典主題已達生產就緒狀態。

:::
