# 📦 經典主題

import APITable from '@site/src/components/APITable';

Docusaurus 的經典主題。

更多配置細節請參閱[主題配置頁面](theme-configuration.mdx)。

```bash npm2yarn
npm install --save @docusaurus/theme-classic
```

:::tip

若您已安裝 `@docusaurus/preset-classic`，則無需將其作為依賴項單獨安裝。

:::

## 配置選項 {#configuration}

可接受字段：

```mdx-code-block
<APITable>
```

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `customCss` | <code>string[] \| string</code> | `[]` | Stylesheets to be imported globally as [client modules](../../advanced/client.mdx#client-modules). Relative paths are resolved against the site directory. |

```mdx-code-block
</APITable>
```

:::note

本主題的多數配置需在 `themeConfig` 中設置，詳見[主題配置](./theme-configuration.mdx)。

:::

### 配置範例 {#ex-config}

可通過預設選項或插件選項配置此主題。

:::tip

多數 Docusaurus 用戶通過預設選項配置此插件。

:::

```js config-tabs
// Preset Options: theme
// Plugin Options: @docusaurus/theme-classic

const config = {
  customCss: './src/css/custom.css',
};
```
