# 📦 theme-mermaid

此主題提供一個由 [mermaid](https://mermaid-js.github.io/) 驅動的 `@theme/Mermaid` 元件。您可以在[圖表](../../guides/markdown-features/markdown-features-diagrams.mdx)文件中閱讀更多相關資訊。

```bash npm2yarn
npm install --save @docusaurus/theme-mermaid
```

## 配置 {#configuration}

```js title="docusaurus.config.js"
export default {
  themes: ['@docusaurus/theme-mermaid'],
  // In order for Mermaid code blocks in Markdown to work,
  // you also need to enable the Remark plugin with this option
  markdown: {
    mermaid: true,
  },
};
```
