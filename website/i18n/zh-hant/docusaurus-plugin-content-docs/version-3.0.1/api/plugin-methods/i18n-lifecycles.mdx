# 國際化生命週期

插件透過這些生命週期來載入與國際化相關的資料。

## `getTranslationFiles({content})` {#getTranslationFiles}

插件宣告其欲使用的 JSON 翻譯檔案。

回傳翻譯檔案 `{path: string, content: ChromeI18nJSON}`：

- `path`：相對於插件本地化資料夾 `i18n/[locale]/[pluginName]` 的路徑。為保持通用性，應省略副檔名 `.json`。
- `content`：使用 Chrome i18n JSON 格式。

這些檔案將由 [`write-translations` CLI](../../cli.mdx#docusaurus-write-translations-sitedir) 寫入插件 i18n 子資料夾，並會在呼叫 [`translateContent()`](#translateContent) 和 [`translateThemeConfig()`](#translateThemeConfig) 前，根據當前語言環境讀取。

範例：

```js title="my-plugin.js"
export default function (context, options) {
  return {
    name: 'my-plugin',
    // highlight-start
    async getTranslationFiles({content}) {
      return [
        {
          path: 'sidebar-labels',
          content: {
            someSidebarLabel: {
              message: 'Some Sidebar Label',
              description: 'A label used in my plugin in the sidebar',
            },
            someLabelFromContent: content.myLabel,
          },
        },
      ];
    },
    // highlight-end
  };
}
```

## `translateContent({content,translationFiles})` {#translateContent}

使用本地化的翻譯檔案來翻譯插件內容。

回傳本地化後的插件內容。

`contentLoaded()` 生命週期將使用 `translateContent()` 回傳的本地化插件內容進行呼叫。

範例：

```js title="my-plugin.js"
export default function (context, options) {
  return {
    name: 'my-plugin',
    // highlight-start
    translateContent({content, translationFiles}) {
      const myTranslationFile = translationFiles.find(
        (f) => f.path === 'myTranslationFile',
      );
      return {
        ...content,
        someContentLabel: myTranslationFile.someContentLabel.message,
      };
    },
    // highlight-end
  };
}
```

## `translateThemeConfig({themeConfig,translationFiles})` {#translateThemeConfig}

使用本地化的翻譯檔案來翻譯網站 `themeConfig` 的標籤。

回傳本地化後的 `themeConfig`。

範例：

```js title="my-plugin.js"
export default function (context, options) {
  return {
    name: 'my-theme',
    // highlight-start
    translateThemeConfig({themeConfig, translationFiles}) {
      const myTranslationFile = translationFiles.find(
        (f) => f.path === 'myTranslationFile',
      );
      return {
        ...themeConfig,
        someThemeConfigLabel: myTranslationFile.someThemeConfigLabel.message,
      };
    },
    // highlight-end
  };
}
```

## `async getDefaultCodeTranslationMessages()` {#getDefaultCodeTranslationMessages}

使用 `<Translate>` API 的主題可提供預設的程式碼翻譯訊息。

應回傳 `Record<string, string>` 格式的訊息，其中鍵為翻譯 ID，值為使用網站當前語言環境本地化的訊息（不含描述）。

範例：

```js title="my-plugin.js"
export default function (context, options) {
  return {
    name: 'my-theme',
    // highlight-start
    async getDefaultCodeTranslationMessages() {
      return readJsonFile(`${context.i18n.currentLocale}.json`);
    },
    // highlight-end
  };
}
```
