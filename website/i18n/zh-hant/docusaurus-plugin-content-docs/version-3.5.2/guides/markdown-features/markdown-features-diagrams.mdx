# 圖表繪製

可使用 [Mermaid](https://mermaid-js.github.io/mermaid/) 程式碼區塊來渲染圖表。

## 安裝方式 {#installation}

```bash npm2yarn
npm install --save @docusaurus/theme-mermaid
```

在 `docusaurus.config.js` 中啟用 Mermaid 功能：添加插件 `@docusaurus/theme-mermaid` 並將 `markdown.mermaid` 設為 `true`。

```js title="docusaurus.config.js"
export default {
  markdown: {
    mermaid: true,
  },
  themes: ['@docusaurus/theme-mermaid'],
};
```

## 使用方式 {#usage}

添加標記為 `mermaid` 語言的程式碼區塊：

````md title="Example Mermaid diagram"
```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```
````

```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```

更多 Mermaid 語法說明請參閱 [Mermaid 語法文件](https://mermaid-js.github.io/mermaid/#/./n00b-syntaxReference)。

## 主題設定 {#theming}

可透過在 `docusaurus.config.js` 的 `themeConfig` 中設定 `mermaid.theme` 值來變更圖表的深色/淺色主題，同時支援為兩種顯示模式分別設定主題。

```js title="docusaurus.config.js"
export default {
  themeConfig: {
    mermaid: {
      theme: {light: 'neutral', dark: 'forest'},
    },
  },
};
```

Mermaid 圖表主題設定詳情請參閱 [Mermaid 主題文件](https://mermaid-js.github.io/mermaid/#/theming)。

## Mermaid 配置 {#configuration}

`mermaid.options` 中的參數將直接傳遞給 `mermaid.initialize`：

```js title="docusaurus.config.js"
export default {
  themeConfig: {
    mermaid: {
      options: {
        maxTextSize: 50,
      },
    },
  },
};
```

可用配置選項請參閱 [Mermaid 配置文件](https://mermaid-js.github.io/mermaid/#/./Setup?id=configuration) 與 [Mermaid 配置類型定義](https://github.com/mermaid-js/mermaid/blob/master/packages/mermaid/src/config.type.ts)。

## 動態 Mermaid 元件 {#component}

欲生成動態圖表，可使用 `Mermaid` 元件：

```mdx title="Example of dynamic Mermaid component"
import Mermaid from '@theme/Mermaid';

<Mermaid
  value={`graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;`}
/>
```
