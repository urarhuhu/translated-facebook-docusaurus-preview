# 📦 plugin-google-tag-manager

import APITable from '@site/src/components/APITable';

用於在 Docusaurus 網站中添加 [Google 代碼管理工具 (gtm.js)](https://developers.google.com/tag-platform/tag-manager) 的外掛程式。可與標準的 [gtag 外掛程式](./plugin-google-gtag.mdx) 搭配使用，深入分析使用者如何使用您的網站。

:::tip

您可以使用 [Google 的標籤助理工具](https://tagassistant.google.com/) 來檢查代碼管理工具是否設定正確！

:::

:::warning[production only]

此外掛程式在開發環境中始終處於非活動狀態，**僅在生產環境中啟用**，以避免污染分析統計數據。

:::

## 安裝 {#installation}

```bash npm2yarn
npm install --save @docusaurus/plugin-google-tag-manager
```

:::tip

如果您使用預設配置 `@docusaurus/preset-classic`，則無需將此外掛程式安裝為依賴項。

您可以透過 [預設選項](../../using-plugins.mdx#docusauruspreset-classic) 來配置此外掛程式。

:::

### 範例配置 {#ex-config}

您可以透過預設選項或外掛程式選項來配置此外掛程式。

:::tip

大多數 Docusaurus 使用者透過預設選項來配置此外掛程式。

:::

```js config-tabs
// Preset Options: googleTagManager
// Plugin Options: @docusaurus/plugin-google-tag-manager

const config = {
  containerId: 'GTM-12345',
};
```
