# 精選資源

精選的 Docusaurus 社群專案列表。

## 社群文件 {#community-documentation}

**[Docusaurus.community](https://docusaurus.community)** 網站延伸了[官方文件](/docs)，提供更複雜的範例和完整文章，推薦最佳實踐並解決常見問題。

它也是一個完整的 Docusaurus 應用程式，就像我們的主站一樣，因此非常適合在那裡託管自訂和實驗範例。

:::info

我們可能會根據情況允許在官方文件中添加指向社群文件的直接連結，如果社群認為這些文章連結有用。

我們也在考慮未來進一步整合到官方文件中。

:::

## 社群插件 {#community-plugins}

### 搜尋 {#search}

- [docusaurus-plugin-lunr](https://github.com/daldridge/docusaurus-plugin-lunr) - 用於建立本地搜尋索引的 Docusaurus 插件，適用於 Lunr.js
- [docusaurus-lunr-search](https://github.com/lelouch77/docusaurus-lunr-search) - Docusaurus 的離線搜尋
- [docusaurus-search-local](https://github.com/cmfcmf/docusaurus-search-local) - Docusaurus 的離線/本地搜尋
- [@easyops-cn/docusaurus-search-local](https://github.com/easyops-cn/docusaurus-search-local) - Docusaurus 的離線/本地搜尋（支援中文）
- [docusaurus-theme-search-typesense](https://github.com/typesense/docusaurus-theme-search-typesense) - 用於 [Typesense DocSearch](https://typesense.org/docs/latest/guide/docsearch.html) 的 Docusaurus 插件
- [@markprompt/docusaurus-theme-search](https://github.com/motifland/markprompt-js/tree/main/packages/docusaurus-theme-search) - 為 Docusaurus 添加生成式 AI / LLM 驅動的文件搜尋，由 [Markprompt](https://markprompt.com) 提供支援。
- [meilisearch-docsearch](https://github.com/tauri-apps/meilisearch-docsearch) - 用於 [Meilisearch](https://www.meilisearch.com) 的 Docusaurus 插件
- [@orama/plugin-docusaurus](https://github.com/askorama/orama/tree/main/packages/plugin-docusaurus) - 適用於 Docusaurus v2 的 [Orama](https://askorama.ai/) 插件
- [@orama/plugin-docusaurus-v3](https://github.com/askorama/orama/tree/main/packages/plugin-docusaurus-v3) - 適用於 Docusaurus v3 的 [Orama](https://askorama.ai/) 插件
- [@getcanary/docusaurus-theme-search-pagefind](https://getcanary.dev/docs/integrations/docusaurus.html) - 建立 [Pagefind](https://pagefind.app/) 索引並使用 [Canary](https://github.com/fastrepl/canary) 作為 UI 基礎元件。

### 整合 {#integrations}

- [docusaurus2-dotenv](https://github.com/jonnynabors/docusaurus2-dotenv) - 支援 dotenv 和其他環境變數的 Docusaurus 外掛
- [posthog-docusaurus](https://github.com/PostHog/posthog-docusaurus) - 將 [PostHog](https://posthog.com/) 產品分析整合至 Docusaurus
- [docusaurus-plugin-moesif](https://github.com/Moesif/docusaurus-plugin-moesif) - 加入 [Moesif API 分析](https://www.moesif.com/) 以追蹤使用者行為，並找出開發者在您的啟用漏斗中流失的環節。
- [docusaurus-plugin-yandex-metrica](https://github.com/sgromkov/docusaurus-plugin-yandex-metrica) - 加入 [Yandex Metrika](https://metrika.yandex.ru/) 計數器，用於評估網站流量和分析使用者行為。

### 功能 {#features}

- [docusaurus-theme-github-codeblock](https://github.com/saucelabs/docusaurus-theme-github-codeblock). 支援從公開 GitHub 儲存庫引用程式碼範例的 Docusaurus 外掛
- [docs-to-pdf](https://github.com/jean-humann/docs-to-pdf) - 將文件轉換為 PDF 格式
- [docusaurus-plugin-sass](https://github.com/rlamana/docusaurus-plugin-sass) - 支援 Sass/SCSS 樣式表
- [docusaurus-plugin-remote-content](https://github.com/rdilweb/docusaurus-plugin-remote-content) - 允許從遠端來源獲取內容的 Docusaurus 外掛
- [@graphql-markdown/docusaurus](https://github.com/graphql-markdown/graphql-markdown) - 從 GraphQL 結構描述生成文件的 Docusaurus 外掛
- [docusaurus-plugin-react-docgen-typescript](https://github.com/atomicpages/docusaurus-plugin-react-docgen-typescript) - 從 `react-docgen-typescript` 載入類型註解的 Docusaurus 外掛
- [docusaurus-plugin-react-docgen](https://github.com/atomicpages/docusaurus-plugin-react-docgen) - 從 `react-docgen` 載入類型註解的 Docusaurus 外掛
- [docusaurus-plugin-module-alias](https://github.com/atomicpages/docusaurus-plugin-module-alias) - 快速為本地模組設定別名的 Docusaurus 外掛
- [docusaurus-protobuffet](https://github.com/protobuffet/docusaurus-protobuffet) - 用於 Protobuf 合約文件的 Docusaurus 工具集
- [docusaurus-prince-pdf](https://github.com/signcl/docusaurus-prince-pdf) - 使用 PrinceXML 生成 PDF，提供更好的字體子集化和目錄功能。支援 Docusaurus 網站
- [redocusaurus](https://github.com/rohit-gohri/redocusaurus) - 將 OpenAPI 文件整合至您的文件的 Docusaurus 預設配置，使用 [Redoc](https://github.com/redocly/redoc)
- [plugin-image-zoom](https://github.com/flexanalytics/plugin-image-zoom) - Docusaurus 的圖片縮放外掛
- [docusaurus-plugin-typedoc](https://github.com/tgreyuk/typedoc-plugin-markdown/tree/master/packages/docusaurus-plugin-typedoc) - 使用 [TypeDoc](https://typedoc.org/) 建立文件的 Docusaurus 外掛
- [docusaurus-openapi-docs](https://github.com/PaloAltoNetworks/docusaurus-openapi-docs) - 用於生成互動式 OpenAPI 文件的 Docusaurus 外掛和主題
- [docusaurus-post-generator](https://github.com/moojing/docusaurus-post-generator) - 讓使用者能透過指令（如 `yarn gen-post new [template] [post_name]`）快速新增部落格/文件的命令列工具。
- [docusaurus-graph](https://github.com/Arsero/docusaurus-graph) - 生成顯示文件檔案間連結的圖形檢視的 Docusaurus 外掛。
- [docusaurus-i18n](https://github.com/moonrailgun/docusaurus-i18n) - 使用 openai 自動翻譯 Docusaurus 文件。
