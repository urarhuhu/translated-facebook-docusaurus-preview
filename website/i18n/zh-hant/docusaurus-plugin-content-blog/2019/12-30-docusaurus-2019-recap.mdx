2019 年對 Docusaurus 來說是豐收的一年——我們在 [Docusaurus 2](https://docusaurus.io/) 上取得了巨大進展。當前未使用翻譯功能的 Docusaurus 1 用戶可隨時試用並[遷移](https://docusaurus.io/docs/migration)至新版！否則我們將在 2020 年協助您完成遷移 :)

## Docusaurus 2 (D2)

2018 年，我們提出要[徹底重構 Docusaurus](https://github.com/facebook/docusaurus/issues/789)。這涉及重大的架構改造——我們從零打造了以內容為核心的 CSS 框架、插件系統，並將靜態 HTML 頁面轉變為具有預渲染路由的單頁應用。這是一場瘋狂的冒險與艱鉅挑戰，尤其在沒有專職全職工程師的情況下。在從實習生成為貢獻者再成為維護者的 [@endilie](https://github.com/endiliey) 協助下，我們在 D2 上取得顯著進展，目前版本為 2.0.0-alpha.40。除翻譯功能外，Docusaurus 1 的所有功能均已移植完成。

D2 的殺手級功能是**深色模式**與**卓越性能**。D2 原生支援深色模式，幾乎無需額外工作即可建立深色模式友好的文檔站點。Endilie 投入大量精力優化站點性能，默認內建多項性能優化技巧——圖像優化、預渲染所有路由為靜態 HTML 後啟用客戶端路由、當用戶懸停導航連結時預取未來導航所需資源等。

最後，我們實現了插件架構並將代碼庫改造成 [Lerna 單體倉庫](https://github.com/facebook/docusaurus/tree/main/packages)。我們相信這種插件架構將有助於社群建設，並讓用戶能為獨特用例構建自訂功能。

## GitHub 動態

- 星標數：10050 → 14632（年增 45.6%）
- 總貢獻者：182 → 303（年增 66.4%），其中多數為非 Facebook 貢獻者
- 日 npm 下載量：728 → 2320（年增 218.7%），峰值出現在 11 月
- 目前 GitHub 上有 3872 個項目使用 D1，247 個項目使用 D2
- 現有 4 位活躍核心貢獻者！（年增 100%）

## 知名用戶

開源社群中相當比例的項目採用 Docusaurus。這半年我們也成功讓更多知名項目遷移至 Docusaurus 2：

- [Create React App](https://create-react-app.dev/)
- [Redux](https://redux.js.org/)
- [Draft.js](https://draftjs.org/)
- [Flux](http://facebook.github.io/flux/)

同時歡迎更多項目加入 Docusaurus 1：

- [Libra](https://developers.libra.org/)
- [MobX](https://mobx.js.org/)
- [Immer](https://immerjs.github.io/immer/)
- [Sorbet](https://sorbet.org/)

## 未來展望

D2 已在[開發者社群](https://docusaurus.io/showcase)中獲得一定關注。2020年，我們的目標是在上半年實現與D1的完整功能對等，並協助剩餘的Facebook專案從D1遷移至D2。若能將Docusaurus用於內部文件也將是重大進展，但這並非易事。若您有此需求或相關想法，歡迎與我們交流！
